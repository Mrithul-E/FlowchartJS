FlowchartJS Documentation

Build interactive, themeable, and performant flowcharts with simple text syntax. Zero dependencies, pure Vanilla JS.

Installation

Get up and running in seconds via CDN or npm.

Playground

Try the library live in your browser.

Configuration

Customize themes, layout, and interactivity.

Shapes

Reference guide for all supported node shapes.

Installation

Include the CSS and JS files in your project:

<link rel="stylesheet" href="dist/flowchart.css">
<script src="dist/flowchart.js"></script>

Basic Usage

Initialize the chart and render content:

const chart = new FlowChart('my-chart');
chart.render('graph TD; A[Start] --> B[End];');

Configuration

Option Type Default Description
theme string 'light' 'light' or 'dark'
interactive boolean true Enable zoom/pan
layout object ... Layout sizing options

Supported Shapes

Use different bracket styles to define node shapes:

[Rect] (Round) ([Stadium]) [[Subroutine]] [(Cylinder)] ((Circle)) {Rhombus} {{Hexagon}} [/Parallelogram/] [\Alt Parallelogram\] [/Trapezoid\] [\Alt Trapezoid/]

API Methods

Method Description
render(input: string) Renders the chart from the given mermaid text input.
setTheme(theme: string) Switches the theme dynamically ('light' or 'dark').
exportImage() Exports the current view as a PNG image.
